# Weather Station - Back-end
Arduino back-end source code.

## What you need ?
- [ ] 1x Arduino Uno card
- [ ] 1x Arduino Ethernet shield
- [ ] 2x DHT11 sensor
- [ ] 1x LCM1 602 IIC V1

## API URLs
- `/data`: Get temperature & humidity
- `/reboot`: Reboot the card
